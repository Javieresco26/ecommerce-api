Record Shop Capstone

Overview

This project is a Record Shop e-commerce application built using Java, Spring Boot, JPA, MySQL, and a frontend website. The application allows users to browse products, search by category and price, log in, and manage a shopping cart.

---Features
View all products

Browse products by category

Filter products by price

User login and authentication

Add products to the shopping cart

Update cart quantities

Remove or clear shopping cart items

Admin category management

---Technologies Used
Java

Spring Boot

Spring Security

Spring Data JPA

MySQL

HTML, CSS, JavaScript

Insomnia

Swagger

---Bugs Fixed
Fixed the category endpoints.

Fixed product search so all matching products are returned.

Fixed product updates so stock quantities are saved correctly.

Implemented shopping cart functionality.
